# Images Directory

## Required Image

Please save your AI JD logo image as `ai-jd-logo.png` in this directory.

The logo should be:
- PNG format with transparent background (recommended)
- Square dimensions (e.g., 512x512px or 1024x1024px)
- High resolution for crisp display

The logo is currently used in:
- Header navigation (16x16 display size, 2x the original)
- Footer (12x12 display size)
